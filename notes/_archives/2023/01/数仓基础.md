## 表命名规范 (非正式)

- 一般表名格式: `prefix_name[_time][_type][_order/version/...]`

| 前缀/后缀 | 说明 |
| --- | --- |
| `dim_` | 维表 |
| `dws_` | 宽表 |
| `fact_` | 事实表 |
| `tmp_` | 临时表 |
| `_bak` | 备份表 |
| `_s` | 快照 (snapshot) |
| `_a` | 全量 (all) |
| `_i` | 增量 (increment) |
| `_si` | 流式增量 (Flowing Increment) |
| `_rt` | 实时 (realtime) |
| `_h/_hr` | 小时 (hour) |
| `_d` | 天 (day) |
| `_w` | 周 (week) |
| `_m` | 月 (month) |
| `_q` | 季 (quarter) |
| `_y` | 年 (year) |
| `_t` | 类型:表 (table) |
| `_v` | 类型:视图 (view) |
