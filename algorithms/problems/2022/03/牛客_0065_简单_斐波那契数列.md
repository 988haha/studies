<summary><b>思路</b></summary>

- `dp(i) := dp(i-1) + dp(i-1)`

<details><summary><b>Python</b></summary>

```python
class Solution:
    def Fibonacci(self , n: int) -> int:
        
        from functools import lru_cache
        
        @lru_cache(maxsize=None)
        def dp(i):
            if i in (1, 2): return 1
            
            return dp(i - 1) + dp(i - 2)
        
        return dp(n)
```
